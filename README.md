## Project Description
A Java desktop application designed for small retail businesses such as sari-sari stores to manage inventory efficiently. 
The system tracks product quantities, monitors expiration dates, and provides simple product insights to help store owners decide which items to restock, prioritize, or phase out. 
It aims to reduce waste, prevent stock shortages, and improve overall store decision-making.

## Proposed Features
- User Login and Register
- Dashboard ( navigation )
- Add Food Item
- Display Inventory ( navigation )
- Insights
- Notifications ( navigation )
- Settings ( navigation )
